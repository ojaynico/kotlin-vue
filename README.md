# kotlin-vue

A simple example of how to use kotlin and vue.

Dependencies used

- kotlinx:kotlinx-html-js
- Vue.js (Added via cdn in src/main/resources/index.html)

Follow the instructions below in your terminal:

1. Change directory into the project directory as below

`cd kotlin-vue`

2. With gradle installed, use the following command to run the application in the browser.

`gradle browserDevelopmentRun`

Your application will be automatically opened in the browser with the url **http:localhost:8080**

If browser does not open, copy the url above and paste it in the browser.
